## :triangular_flag_on_post: How to use:

1. [Download the ZIP](https://github.com/w1nte/VSC-DLV-Syntax-Highlighting/archive/main.zip).
2. Unpack it into `<user home>/.vscode/extensions`
3. Restart VSCode.

**Enjoy! :yum:**
